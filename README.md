## Installation

Run the following command in a terminal and you're ready to go!

**Windows**
```powershell
iwr instl.sh/x0f5c3/tl-dl/windows | iex
```

**macOS**
```bash
curl -sSL instl.sh/x0f5c3/tl-dl/macos | bash
```

**Linux**
```bash
curl -sSL instl.sh/x0f5c3/tl-dl/linux | bash
```
